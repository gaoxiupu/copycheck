# PagePilot

PagePilot 是一款基于 AI 的浏览器插件，旨在帮助网站建设服务商、QA 团队和内容编辑人员快速走查网页内容。通过集成多种先进的 AI 模型，PagePilot 可以一键检查页面上的拼写错误、排版问题、文本一致性以及其他影响用户体验的问题，并生成详细的检查报告。

## 核心功能

- **智能内容检查**：自动识别页面可见文本，包括标题、段落、按钮、标签等。
- **多维度分析**：
  - **拼写检查**：准确识别中英文拼写错误。
  - **排版建议**：检测中英文混排空格、符号使用不规范等问题。
  - **一致性检查**：发现同一功能或术语在页面中的表述差异。
  - **体验优化**：识别语义模糊或表述不当的文案。
- **多模型支持**：支持智谱AI (GLM)、DeepSeek、阿里通义 (Qwen)、Google Gemini 等主流 AI 服务。
- **实时进度与报告**：检查过程中实时展示进度，完成后提供结构化的检查报告。
- **报告导出**：支持导出为 CSV 格式，方便团队协作与问题跟踪。
- **历史记录管理**：自动保存最近的检查记录，随时回溯。

## 安装指南

1. 下载或克隆本项目代码。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目中的 `src` 目录。
5. 插件图标将出现在浏览器工具栏中。

## 使用说明

1. **配置 API Key**：
   - 点击插件图标打开侧边栏。
   - 进入设置页面，选择您偏好的 AI 服务商。
   - 输入相应的 API Key 并保存配置。
2. **开始检查**：
   - 导航至需要检查的网页。
   - 点击插件中的“开始检查当前页面”按钮。
   - 等待 AI 分析完成。
3. **处理结果**：
   - 在结果列表中查看发现的问题及其严重程度。
   - 参考 AI 提供的修改建议。
   - 根据需要导出报告。

## 技术架构

- **前端框架**：原生 JavaScript/HTML/CSS (采用 Chrome Extension Manifest V3 标准)。
- **AI 集成**：通过 Fetch API 访问各服务商的 REST 接口。
- **本地存储**：使用 `chrome.storage.local` 安全存储用户配置和历史记录。

## 隐私与安全

- 您的 API Key 仅保存在本地浏览器的存储中。
- 只有在您点击“开始检查”时，当前页面的可见文本才会发送至您选择的 AI 服务商进行分析。
- 插件不会收集您的个人浏览历史或敏感信息。
