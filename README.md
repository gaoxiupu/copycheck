## 🌏 中文说明

CopyCheck 旨在帮助 **QA 团队**、**内容运营**及**前端开发者**高效完成网页上线前的“文案走查”。通过集成顶级大语言模型，它能精准识别那些肉眼难以察觉的细微错误。

### ✨ 核心特性

-   **🎯 多维度智能审计**：
    -   **拼写与术语**：识别中英文错别字、专有名词大小写（如 `iPhone` vs `iphone`）。
    -   **排版规范**：严格执行“盘古规范”（中英文间距）、全半角标点校验。
    -   **一致性检查**：确保页面内人称（你/您）、单位、操作指引统一。
    -   **语境适用性**：评估按钮文案的简洁度、链接的描述性及逻辑严谨性。
-   **🤖 多模型无缝切换**：支持 **Google Gemini**, **DeepSeek**, **智谱 AI (GLM)**, **阿里通义千问 (Qwen)**，以及任何 **OpenAI 兼容接口**。
-   **⚙️ 自定义规则**：允许用户添加特定业务逻辑或品牌风格指南作为额外的审计标准。
-   **📊 结构化报告**：提供“严重、中等、轻微”分级的详细错误列表，并给出修改建议。
-   **📈 高性能分析**：支持超长网页的分段同步处理，实时展示分析进度。
-   **📥 导出与协作**：支持 CSV 报告导出，方便团队协同修复。

### 🚀 快速开始

1.  **安装扩展**：
    -   克隆仓库：`git clone https://github.com/gaoxiupu/copycheck.git`
    -   打开 Chrome/Edge `chrome://extensions/` 页面。
    -   开启“开发者模式”。
    -   点击“加载已解压的扩展程序”，选择 `src` 目录。
2.  **配置 API**：
    -   在插件侧边栏的“设置”中选择 AI 提供商并填入 API Key。
3.  **开始审计**：
    -   在任意网页点击扩展图标，打开侧边栏。
    -   点击“开始检查”，即可查看实时分析结果。

### 🛠️ 技术栈

-   **Manifest V3**: 现代浏览器扩展标准。
-   **Vanilla JS/HTML/CSS**: 无重度框架，轻量且高性能。
-   **Chrome SidePanel API**: 提供沉浸式的侧边栏交互体验。

## 🌐 English

CopyCheck is an AI-driven browser extension designed for meticulous web content auditing. It helps teams ensure their website copy is professional, consistent, and error-free.

### 🌟 Key Features

-   **Multi-dimensional Audit**: Checks for spelling, typography (Pangu spacing), consistency, and contextual relevance.
-   **Modern AI Support**: Integrated with Gemini, DeepSeek, Zhipu GLM, Qwen, and custom OpenAI APIs.
-   **Severity Scoring**: Issues are categorized into Critical, Medium, and Minor for better prioritization.
-   **Custom Rules**: Define your own brand voice or specific constraints for the AI to follow.
-   **Optimized Performance**: Handles long-form content via smart text chunking and provides real-time progress.

### 📥 Installation

1. Clone this repository.
2. Go to `chrome://extensions/` in your browser.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the `src` folder.
